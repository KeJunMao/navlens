## 简介

一个开源导航网站，速度非常快

## 特性

- 😃 [100,000+](https://icones.js.org/) 图标可供使用
- ⚡️ Nuxt3 驱动，就是快！
- 🔥 PWA 当然

## 部署

```bash
git clone https://github.com/KeJunMao/navlens.git
docker compose build
docker compose up
```

## 💻 发展

- 克隆此代码库
- 使用 `corepack enable` 启用 Corepack (对于 Node.js < 16.10，请使用`npm i -g corepack`)
- 使用 `pnpm install` 安装依赖项
- 使用 `pnpm dev` 运行交互测试
